Was gute Meta-Tags leisten

Der Seitentitel sagt Suchmaschinen und Menschen, worum es auf einer Seite geht. Die Description ergänzt den Gedanken und kann im Suchergebnis als Beschreibung erscheinen. Beides sollte die konkrete Seite treffen, nicht die ganze Website in allgemeine Werbesätze pressen.

Canonical und Robots

Der Canonical-Link benennt die bevorzugte URL. Das ist wichtig, wenn derselbe Inhalt über mehrere Adressen erreichbar ist. Die Robots-Angabe steuert, ob eine Seite indexiert werden soll und ob Crawler ihren Links folgen dürfen. Eine versehentliche noindex-Angabe kann eine Seite aus Suchergebnissen entfernen.

Open Graph und Twitter Card

Diese Angaben prägen die Vorschau, wenn ein Link in sozialen Netzwerken oder Messengern geteilt wird. Die Plattform kann trotzdem eine eigene Darstellung wählen oder bereits gespeicherte Daten zeigen.

Häufige Fragen

Wie lang sollte ein Seitentitel sein?
Eine feste Zeichengrenze gibt es nicht, weil Suchmaschinen nach verfügbarer Breite kürzen. Etwa 50 bis 60 Zeichen sind oft eine brauchbare Orientierung, aber kein Qualitätsbeweis.
Garantieren gute Meta-Tags bessere Platzierungen?
Nein. Sie machen Inhalt und Vorschau verständlicher, ersetzen aber weder nützliche Seiteninhalte noch Technik, interne Verlinkung, Autorität oder eine klare Suchabsicht.
Warum wird meine Vorschau später anders dargestellt?
Suchmaschinen und soziale Plattformen dürfen Texte kürzen, andere Seitenausschnitte wählen oder alte Angaben aus ihrem Cache zeigen. Der Generator bildet eine realistische Orientierung ab, keine verbindliche Plattformausgabe.
