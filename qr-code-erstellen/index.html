QR-Code kostenlos erstellen

Erzeuge einen dauerhaften QR-Code für einen Link, freien Text, WLAN-Zugang oder Kontaktdaten. Als PNG für den Alltag oder als scharfes SVG für den Druck.

Keine WeiterleitungDer Code enthält deine Daten direkt. Kein Tracking-Link liegt dazwischen.

Ein QR-Code, der niemandem gehört außer dir

Viele kostenlose Generatoren legen zwischen QR-Code und Zieladresse einen eigenen Kurzlink. Der Anbieter kann Aufrufe messen, das Ziel später sperren oder für eine weitere Nutzung Geld verlangen. Dieses Werkzeug schreibt den Inhalt unmittelbar in den Code. Es gibt kein Konto und keinen Dienst, von dem der QR-Code später abhängt.

Für Bildschirm und Druck

PNG ist unkompliziert und funktioniert in nahezu jedem Programm. SVG ist eine Vektordatei: Sie bleibt auch auf einem Plakat oder Schild scharf. Prüfe den fertigen Code vor dem Druck immer mit mehreren Handys und lasse rundherum ausreichend freie Fläche.

Häufige Fragen

Läuft der QR-Code irgendwann ab?
Nein. Er enthält den Link oder die Angaben direkt. Solange der enthaltene Link erreichbar bleibt, funktioniert auch der Code.
Kann jemand mein WLAN-Passwort aus dem Code lesen?
Ja. Ein QR-Code verschlüsselt den Inhalt nicht, sondern stellt ihn nur maschinenlesbar dar. Gib einen WLAN-Code deshalb nur an Personen weiter, die das Passwort kennen dürfen.
Warum sollte der Code dunkel auf hellem Grund sein?
Scanner erkennen klare, kontrastreiche Module am zuverlässigsten. Sehr helle Vordergrundfarben, transparente Hintergründe und zu wenig freie Fläche können das Scannen erschweren.
