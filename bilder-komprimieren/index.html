Große Bilddateien bremsen Websites, füllen Postfächer und kosten auf dem Handy unnötig Datenvolumen. Die Bildwerkstatt setzt deshalb an zwei Stellen an: Sie reduziert die Pixelmaße und speichert das Ergebnis mit einer passenden Kompression neu.

WebP, JPG, PNG oder AVIF?

  • WebP ist eine gute Standardwahl für Fotos und Grafiken im Web.
  • JPG eignet sich für Fotos, wenn möglichst viele Programme das Ergebnis öffnen sollen.
  • PNG bewahrt transparente Flächen und scharfe Grafiken, erzeugt aber oft größere Dateien.
  • AVIF kann besonders kleine Dateien liefern. Die Ausgabe erscheint nur, wenn dein Browser AVIF selbst erzeugen kann.

Was mit Metadaten passiert

Das Bild wird auf einer neuen Browser-Zeichenfläche aufgebaut. EXIF-Daten, Kameramodell, Aufnahmezeit und gespeicherte GPS-Koordinaten werden nicht in die neue Datei übernommen.

Häufige Fragen

Werden meine Bilder hochgeladen?
Nein. Die Website lädt nur den Programmcode. Deine Bilder werden danach im Browser auf deinem Gerät gelesen, verarbeitet und wieder zum Herunterladen bereitgestellt.
Warum wird ein Bild manchmal größer?
PNG und sehr hohe Qualitätsstufen können größere Dateien erzeugen als das Original. Wähle für Fotos WebP oder JPG und reduziere die Qualität schrittweise.
Funktioniert das auch auf dem Handy?
Ja. Bei sehr großen oder vielen Bildern kann ein Handy jedoch schneller an seine Speichergrenze kommen. Dann ist eine kleinere Auswahl pro Durchgang sinnvoll.
