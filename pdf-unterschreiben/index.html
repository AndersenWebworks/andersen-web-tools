Eine sichtbare Unterschrift, ohne das Dokument wegzugeben

Viele Formulare brauchen nur eine eingescannte oder gezeichnete Unterschrift an einer bestimmten Stelle. Dafür musst du die PDF nicht auf einen fremden Server laden: Seite auswählen, Unterschrift platzieren, Größe prüfen und die neue Datei herunterladen.

Was dieses Werkzeug genau macht

Die Unterschrift wird als Bild in die ausgewählte PDF-Seite eingesetzt. Das Ergebnis sieht in normalen PDF-Programmen aus wie ein unterschriebenes Dokument. Es enthält jedoch kein Zertifikat und keine technische Identitätsprüfung.

Original behalten

Die Quelldatei wird nicht verändert. Bewahre sie auf, falls du die Position später korrigieren oder eine zertifikatsbasierte Signatur ergänzen möchtest.

Häufige Fragen

Ist die Unterschrift rechtsverbindlich?
Das hängt vom konkreten Dokument und der vereinbarten Form ab. Das Werkzeug setzt nur das sichtbare Unterschriftsbild ein; es entscheidet nicht, welche Signaturform für deinen Fall erforderlich ist.
Wird meine Unterschrift gespeichert?
Nein. Sie liegt nur vorübergehend im Arbeitsspeicher des Browsers und steht nach Schließen oder Zurücksetzen der Seite nicht mehr zur Verfügung.
Kann ich auf mehreren Seiten unterschreiben?
In einem Durchgang wird eine Unterschrift auf einer Seite eingesetzt. Für eine weitere Stelle kannst du die erzeugte PDF erneut öffnen.
